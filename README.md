# credal-cpd


## Requirements:
- Python
- numpy
- pandas
- pgmpy
- scikit-learn
- tqdm
- joblib

## Files
- `train.py` trains 5 models for each of the 14 data sets 
- `eval.py` computes accuracy, uncertainty, and discounted-accuracy for each of the models 
- `print_results.py` prints 3 tables for each of the metrics
